# Security Policy

لا تنشر تفاصيل أمنية حساسة أو بيانات مصادر بشرية في Issue عامة. أرسل بلاغاً خاصاً إلى مالك المستودع مع وصف الأثر وخطوات التحقق الآمنة.

يشمل النطاق الموقع الثابت، إعدادات Hugo، البيانات العامة، وسير عمل GitHub Actions. لا تُحفظ التحقيقات غير المنشورة أو هويات المصادر في هذا المستودع العام.
